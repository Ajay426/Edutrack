# 📚 EduTrack – Attendance Management System

EduTrack is a Full Stack Attendance Management System that simplifies attendance tracking and management for educational institutions. It provides a secure, role-based authentication system with separate dashboards for Admins, Faculty, and Students, ensuring efficient attendance management and easy access to records.

## 🚀 Features

- 🔐 Role-Based Authentication & Authorization
- 👨‍🏫 Faculty Dashboard
- 🎓 Student Dashboard
- 👨‍💼 Admin Dashboard
- ✅ Mark & Manage Attendance
- 📊 Attendance Reports & History
- 👥 Student Management
- 🏫 Course/Class Management
- 📱 Responsive User Interface
- 🔒 Secure Login System

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/Edutrack.git
```

### Navigate to the project

```bash
cd Edutrack
```

### Install dependencies

Frontend

```bash
cd FRONTEND
npm install
npm start
```

Backend

```bash
cd BACKEND
npm install
npm run dev
```

## 🎯 Project Objective

The objective of EduTrack is to automate the attendance process, reduce manual record-keeping, improve accuracy, and provide a centralized platform for students, faculty, and administrators to manage attendance efficiently.
